# Una sorpresa para ti 🪼🪷

Una carta interactiva en HTML con:

- ✉️ Sobre con sello de cera animado
- 🔥 Animación de la carta saliendo y quemándose con brasas
- 🖋️ Tipografía manuscrita antigua (Great Vibes + Cormorant Garamond)
- 📜 Aspecto de papel viejo con bordes quemados y líneas de libreta
- 🪼 Mensajes secretos al hacer clic en medusas y lotos
- 🎮 Minijuego: encuentra las 5 medusas escondidas
- ✨ Partículas flotantes de fondo
- ❦ Mensaje final con ornamentos

## Cómo abrir

Solo haz doble clic en `carta-interactiva.html` y se abrirá en tu navegador. No necesita servidor.

## Cómo desplegar en GitHub Pages (opcional)

Para que esté disponible online, en el repo ir a **Settings → Pages** y seleccionar la rama `main` como fuente.
